✈️ Full-Stack Airline Booking App
A modern, scalable, and secure full-stack airline booking system built with Spring Boot (Backend), React (Frontend), and React Native (Mobile).

🚀 Project Overview
This project is a complete airline reservation platform that allows users to search flights, select seats, manage bookings, and interact with a fully secure backend system.
It demonstrates real-world full-stack architecture, including authentication, role-based authorization, and dynamic UI experiences.


⚙️ Backend (Spring Boot)
🔧 Core Technologies
  •Java
  •Spring Boot
  •Spring Security
  •JWT (JSON Web Tokens)
  •JPA / Hibernate
  •PostgreSQL
  •JavaMailSender

🔥 Features

🧩 RESTful API Design
  •Clean and scalable API architecture
  •Endpoints for flights, bookings, users, and authentication
  
🗄️ Database Management
  •Integration with PostgreSQL
  •Entity relationships and persistence using JPA
  
🔐 Security & Authentication
  •Spring Security implementation
  •JWT-based authentication (stateless)
  •Secure API communication
  
👥 User & Role Management
  •Role-based system:

    •Pilot

📧 Email Integration
